Welcome

If you want to see all of the bank's data , go to this URL:

/api/v1/bank

If you want to see a specific user info , go to this URL:

/api/v1/bank/ [id of the user]

Filter users by cash higher/equal than x amount?

/api/v1/bank/filter-cash/more-than?cash= [x amount]

Filter users by cash lower/equal than x amount?

/api/v1/bank/filter-cash/less-than?cash= [x amount]
